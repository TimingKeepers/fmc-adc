RESERVED[3:0] DDR3_CAL_DONE SYS_PLL_LCK P2L_PLL_LCK FMC_PRES
  • FMC_PRES [read-only]: FMC presence
    0: FMC slot is populated
    1: FMC slot is not populated.
  • P2L_PLL_LCK [read-only]: GN4142 core P2L PLL status
    0: not locked
    1: locked.
  • SYS_PLL_LCK [read-only]: System clock PLL status
    0: not locked
    1: locked.
  • DDR3_CAL_DONE [read-only]: DDR3 calibration status
    0: not done
    1: done.
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's.

    Manual control of the front panel red LED (unused in the fmc-adc application)

    Active low clear signal for VCXO DACs
