3. Register description

3.1. Control register

HW prefix: fmc_adc_core_ctl
HW address: 0x0
C prefix: CTL
C offset: 0x0

RESERVED[23:16]
23 22 21 20 19 18 17 16

7 6 5 4 3 2 1 0
ACQ_LED TRIG_LED TEST_DATA_EN MAN_BITSLIP OFFSET_DAC_CLR_N FMC_CLK_OE FSM_CMD[1:0]
  • FSM_CMD [read/write]: State machine commands
    1: start
    2: stop
  • FMC_CLK_OE [read/write]: FMC Si750 output enable
  • OFFSET_DAC_CLR_N [read/write]: Offset DACs clear (active low)
  • MAN_BITSLIP [write-only]: Manual serdes bitslip
  • TEST_DATA_EN [read/write]: Enable test data
    Write the address counter value instead of ADC data to DDR
  • TRIG_LED [read/write]: Manual TRIG LED
    Manual control of the front panel TRIG LED
  • ACQ_LED [read/write]: Manual ACQ LED
    Manual control of the front panel ACQ LED
  • RESERVED [read/write]: Reserved
    Ignore on read, write with 0's

  • FSM [read-only]: State machine status
    States:
    1: IDLE
    2: PRE_TRIG
    3: WAIT_TRIG
    4: POST_TRIG
    5: DECR_SHOT
    7: Illegal
  • SERDES_PLL [read-only]: SerDes PLL status
  • SERDES_SYNCED [read-only]: SerDes synchronization status
    0: bitslip in progress
    1: serdes synchronized
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's

3.3. Trigger configuration

HW prefix: fmc_adc_core_trig_cfg
HW address: 0x2
C prefix: TRIG_CFG
C offset: 0x8

INT_TRIG_THRES[15:8]
23 22 21 20 19 18 17 16
INT_TRIG_THRES[7:0]
15 14 13 12 11 10 9 8
RESERVED[9:2]
7 6 5 4 3 2 1 0
RESERVED[1:0] INT_TRIG_SEL[1:0] SW_TRIG_EN HW_TRIG_EN HW_TRIG_POL HW_TRIG_SEL
  • HW_TRIG_SEL [read/write]: Hardware trigger selection
    0: internal (data threshold)
    1: external (front panel trigger input)
  • HW_TRIG_POL [read/write]: Hardware trigger polarity
    0: positive edge/slope
    1: negative edge/slope
  • HW_TRIG_EN [read/write]: Hardware trigger enable
    0: disable
    1: enable
  • SW_TRIG_EN [read/write]: Software trigger enable
    0: disable
    1: enable
  • INT_TRIG_SEL [read/write]: Channel selection for internal trigger
    00: channel 1
    01: channel 2
    10: channel 3
    11: channel 4
  • RESERVED [read/write]: Reserved
    Ignore on read, write with 0's
  • INT_TRIG_THRES [read/write]: Threshold for internal trigger
    Treated as binary two's complement and compared to raw ADC data

RESERVED[0:0] SSR[6:0]
  • SSR [read/write]: Solid state relays control for channel 1
    Controls input voltage range, termination and DC offset error calibration
    0x23: 100mV range
    0x11: 1V range
    0x45: 10V range
    0x00: Open input
    0x42: 100mV range calibration
    0x40: 1V range calibration
    0x44: 10V range calibration
    Bit3 is indepandant of the others and enables the 50ohms termination.
  • RESERVED [read/write]: Reserved
    Ignore on read, write with 0's
